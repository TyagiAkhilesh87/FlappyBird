## ğŸ•¹ï¸ Getting Started

### Prerequisites
- Node.js & npm
- Expo Go (for mobile testing)

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/TyagiAkhilesh87/FlappyBird.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up `.env`:
   ```env
   EXPO_PUBLIC_SUPABASE_URL=your_supabase_url
   EXPO_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
4. Start the engine:
   ```bash
   npx expo start -c
   ```
